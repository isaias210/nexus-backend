# Nexus Backend

## Installation
npm install

## Run locally
npm start

## Deploy on Render
1. Push this repo to GitHub
2. Create new Web Service on Render
3. Connect GitHub repo
4. Start command: npm start
5. Add environment variables in Render dashboard

## Configure OAuth Redirect URIs

Instagram:
https://YOUR_RENDER_DOMAIN/oauth/instagram/callback

YouTube:
https://YOUR_RENDER_DOMAIN/oauth/youtube/callback
